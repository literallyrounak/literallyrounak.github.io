# Brutalist Portfolio

A personal portfolio website built with React and pure CSS, embracing sophisticated brutalist web design principles.

## Features

- Raw, high-contrast brutalist aesthetic with thick borders and heavy shadows
- Responsive grid layouts for projects and skills
- Oversized typography using Space Mono and Inter fonts
- Hand-crafted hover interactions and push effects
- Simple, semantic structure with no external styling frameworks
- Framer Motion for subtle entrance animations
